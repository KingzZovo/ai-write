## 总体思路

v1.5.0 已交付 4 大主线（C1 scene-staged / C2 auto-revise / C3 prompt cache / C4 cascade）。v1.5.x + v1.6 只做**运营面与上游依赖优化**，不引入架构变更。

- v1.5.x 是接在 v1.5.0 后的收尾补丁（只走补丁版本号，不走 minor）。
- v1.6.0 是下一个 minor，聈践上游 LLM provider prompt cache + scene_mode 观测面 + cascade UI。
